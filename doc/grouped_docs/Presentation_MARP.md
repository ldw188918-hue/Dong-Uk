# BizInsight Local: 소상공인을 위한 AI 비즈니스 파트너
> **MARP 프레임워크 기반 프레젠테이션 (Presentation)**

---

## 1. Reason (배경 및 문제 제기)
**"왜 이 서비스가 필요한가?"**

### 🚩 현실 (Reality)
*   **데이터의 부재가 아닌, 활용의 부재**: 수많은 소상공인(식당, 카페 등)이 POS 기기를 통해 매일 방대한 매출 데이터를 쌓고 있지만, 이를 엑셀 파일 형태로 보관만 할 뿐 실제 경영에 활용하지 못하고 있습니다.
*   **높은 진입 장벽**: 전문적인 데이터 분석 툴은 너무 어렵고, 데이터 분석가를 고용할 여력은 없습니다.
*   **보안 우려**: 매출 데이터는 민감한 정보이기에 클라우드 기반의 외부 AI 서비스(ChatGPT 등)에 섣불리 업로드하기 꺼려집니다.

### 🎯 기회 (Opportunity)
*   "복잡한 설정 없이, 내 PC에서 안전하게, 전문가 수준의 인사이트를 얻을 수는 없을까?"라는 질문에서 시작했습니다.

---

## 2. Approach (해결 방안 및 핵심 전략)
**"우리는 이 문제를 어떻게 해결하는가?"**

### 💡 솔루션: BizInsight Local
*   **제품 정의**: 로컬 LLM(Local Large Language Model)을 활용한 **PC 설치형 비즈니스 인사이트 대시보드**입니다.

### 🔑 핵심 기능 (Key Features)
1.  **Zero-Configuration**: 복잡한 설정 없이 엑셀 파일을 드래그 앤 드롭(Drag & Drop)하면 즉시 분석이 시작됩니다.
2.  **Privacy First (Local AI)**: 데이터가 외부 서버로 전송되지 않습니다. Ollama와 연동하여 사용자의 로컬 환경에서 모든 분석과 추론이 이루어집니다.
3.  **Actionable Insight**: 단순한 그래프 나열이 아닙니다.
    *   *Before*: "금요일 매출 100만원"
    *   *After*: **"금요일 저녁 재고를 1.5배 더 확보하세요. 지난 3주간 금요일마다 재료 소진이 발생했습니다."**

### 🛠 기술 스택 (Tech Stack)
*   **Core**: Python 3.9+
*   **UI/UX**: Streamlit (직관적인 인터랙티브 대시보드)
*   **AI Engine**: Ollama (Llama 3, Mistral 등 고성능 경량 모델 활용)

---

## 3. Result (기대 효과 및 가치)
**"사용자는 무엇을 얻는가?"**

### ✨ 사용자 가치 (User Value)
*   **데이터 기반 의사결정**: "감"에 의존하던 경영에서 벗어나, 데이터 수치에 기반한 정확한 판단이 가능해집니다.
*   **시간 절약**: 매일 장부 정리에 들이던 시간을 획기적으로 단축하고, 본업인 서비스와 품질 향상에 집중할 수 있습니다.
*   **매출 증대**: 구체적인 타임 세일 전략, 메뉴 최적화 제안 등을 통해 실질적인 매출 상승을 기대할 수 있습니다.

### 🔒 차별화된 강점
*   **안전함**: 인터넷이 끊겨도 작동하며, 내 소중한 영업 비밀이 100% 보호됩니다.
*   **쉬움**: 개발 지식이 전혀 없는 사장님도 바로 사용할 수 있는 직관성을 제공합니다.

## 4. Plan (향후 계획 및 로드맵)
**"앞으로 어떻게 발전할 것인가?"**

### 🚀 Roadmap
*   **Phase 1 (MVP - 현재)**
    *   CSV/Excel 파일 업로드 및 시각화 대시보드 구현.
    *   기본적인 통계 기반 텍스트 요약 및 LLM 연동 검증.
*   **Phase 2 (v1.1 - 고도화)**
    *   여러 달의 매출 파일 병합 분석 기능 추가.
    *   업종별(카페, 식당, 소매) 맞춤형 프롬프트 튜닝.
*   **Phase 3 (v2.0 - 확장)**
    *   **예측(Forecasting)**: "다음 달 예상 매출" 시계열 예측 모델 탑재.
    *   PDF 리포트 생성 및 배포 패키지(exe) 최적화.
